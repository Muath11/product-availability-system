# Product Availability System

A frontend project built using HTML, CSS, JavaScript, and Firebase.

## Description
This project includes two separate interfaces:
- **Client Page**: Displays product availability status to customers.
- **Employee (Admin) Page**: Allows employees to update product availability in real-time.

Firebase is used to store and sync product availability data.

## Features
- Client & Admin role separation
- Real-time product availability updates
- Firebase Firestore integration
- Clean and responsive UI
- Deployed using Netlify

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase (Firestore)
- Netlify
